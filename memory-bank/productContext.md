# Product Context

## Why This Project Exists
Weyoto Voice exists to provide a simple, powerful way to convert text into natural-sounding speech. It eliminates the need for microphones, recording equipment, or complex audio editing software.

## How It Works
1. User enters text they want converted to speech
2. User selects a voice and optionally provides tone instructions
3. The backend sends the request to OpenAI's TTS API
4. The generated audio is streamed back and saved as an MP3
5. User can download or listen to the result

## User Experience Goals
- **Simplicity**: Minimal clicks to go from text to speech
- **Speed**: Streaming generation means near-instant results
- **Quality**: Natural-sounding voices with tone customization
- **Accessibility**: Clean, responsive UI that works on any device
- **Professionalism**: Polished design that inspires trust

## Target Users
- Content creators needing voiceovers
- Developers prototyping voice features
- Storytellers and narrators
- Accessibility use cases
- Anyone needing quick, high-quality speech synthesis
